

### Summary ############

This assignment consists of building Vending Machine with all possible tests

        Accepts coins of 1,5,10,25 Cents i.e. penny, nickel, dime, and quarter.
        Allow user to select products Coke(25), Pepsi(35), Soda(45)
        Allow user to take refund by cancelling the request.
        Return selected product and remaining change if any.
        Allow reset operation for vending machine supplier.


###### Assumptions ##############

This program allows only 1 product to be selected/purchased at one point of time.
System used Windows
Editor used : IntelliJ

###### Tests contents details ##############

1. Product_Main.java: This consist of main program where all the operations of Vending Machine documented as follows:
 	 0 - To print instruction.
	 1 - Print Product List
	 2 - To select item to purchase from Product List
	 3 - To print selections
	 4 - To make Payment
	 5 - To Cancel item
	 6 - To ResetMachine
	 7 - Exit.
 
2. Products.java : this contain all the functions, which called to perform the abve mentioned operation of Vending Machine. 
3. ProductFactory.java: this creates the Products along with the prices.

######### Quick Start############

1. Install IntelliJ.
2. Open Project
3. Run Product_Main.java
